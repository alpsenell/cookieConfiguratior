# Cookie configurator
### Bundle build is under the `dist/` directory.

#### Project setup
```
npm install
```

#### Compiles and hot-reloads for development
```
npm run serve
```

#### Compiles and minifies for production
```
npm run build
```

#### For running tests
```
npm run test:unit
```

#### For linting;
```
npm run lint
```
